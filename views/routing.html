Highlights

  • angular-route.js provides a module called ngRoute, which does routing and deep-linking services and directives for AngularJS apps.
  • Routing is taken care by an AngularJS Provider called $routeProvider.
  • Add depedecy on ngRout to the main module, which is vinceApp.
  • define routeConfigurator() function, where the mappings are defined using the provider for the $route service, $routeProvider.
  • call config() on module, passing routeConfigurator
  • ng-view combined with route together, you can implement navigation
  • When the value returned by the $location.path method matches one of the mappings, the corresponding view file will be loaded and displayed.
  • $routeProvider accepts two arguments, the second is a config object, you have to provide its templateUrl.
  • add url path to anchor tag: #/path
  • or call $location.path() to trigger navigation (you will need to add the dependency on $location).
  • $location supports different modes, you can config it in routeConfigurator()
