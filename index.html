Highlights

  1. Module is container of reusable components, it is defined by using angular.module()
  2. One html page can be assigned only one module, via ng-app directive on html tag
  3. app.js is the entry point of the js part of the application
  4. ng-include is used to load html fragements from external html file, it limits the file to be in the same domain
